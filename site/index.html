Tính năng

Mọi thứ bạn cần

Từ dev nhanh đến production-grade self-host — htn-tunnel lo hết.

⚡

Dùng ngay, không đăng ký

Chạy htn-tunnel http 3000 là xong. Không tài khoản, không cấu hình phức tạp.

🌐

HTTP & TCP Tunnels

HTTPS subdomain routing cho web app. Raw TCP cho database, SSH, bất kỳ port nào.

🔐

Browser Login

htn-tunnel login mở browser tự động. Đăng ký xong, key tự lưu — không cần copy paste.

📌

Subdomain cố định

Claim một lần, dùng mãi. myapp.33.id.vn luôn trỏ về máy bạn khi tunnel online.

📊

Live Analytics Dashboard

Request log real-time, traffic charts, status breakdown HTTP. Mở ngay bằng htn-tunnel dashboard.

🔒

Auto-TLS (Let's Encrypt)

Wildcard certificate tự động qua DNS-01 Cloudflare. Không cần cấu hình SSL thủ công.

🔌

WebSocket Support

Full proxy với Origin rewrite. HMR, dev server, Socket.IO — tất cả hoạt động ngay.

🛠️

Self-Host Wizard

htn-server init — interactive setup, tự cấu hình nginx, SSL, systemd. Zero manual config.

♻️

Auto-upgrade & Backup

Tự update từ GitHub Releases. Backup/restore dữ liệu để migrate server dễ dàng.

Cài đặt

Một lệnh là xong

Yêu cầu Node.js 16+. Hỗ trợ Windows, macOS, Linux.

$ npm install -g htn-tunnel

Hoặc tải binary thẳng từ GitHub Releases (không cần Node.js), hoặc go install github.com/nhh0718/htn-tunnel/cmd/htn-tunnel@latest

Hướng dẫn

Quick Start

Từ cài đặt đến tunnel đầu tiên — chưa đến 2 phút.

1

Cài đặt

Cài htn-tunnel qua npm (Node.js 16+ required)

# Cài đặt toàn cục
npm install -g htn-tunnel
2

Kết nối server (chỉ làm 1 lần)

Nhập địa chỉ server tunnel. Dùng server công cộng hoặc self-host của bạn.

htn-tunnel setup
# Nhập địa chỉ server: 33.id.vn:4443
3

Tunnel ngay — không cần đăng ký

Expose port local ra internet trong vài giây. Subdomain random, có hiệu lực 2 giờ.

htn-tunnel http 3000
# ✓ Tunnel online
# → https://abc12xyz.33.id.vn
4

Đăng ký để có subdomain cố định (tuỳ chọn)

Login qua browser, claim subdomain theo ý muốn — dùng vĩnh viễn.

# Đăng ký / đăng nhập — mở browser tự động
htn-tunnel login

# Tunnel với subdomain cố định
htn-tunnel http 3000:myapp
# → https://myapp.33.id.vn  (vĩnh viễn)
CLI Reference

Tất cả lệnh

Danh sách đầy đủ các lệnh htn-tunnel client.

Lệnh Mô tả Khi nào dùng
htn-tunnel setup Nhập địa chỉ server tunnel Lần đầu tiên
htn-tunnel login Đăng ký / đăng nhập qua browser, key tự lưu Khi muốn subdomain cố định

htn-tunnel logout Xóa API key đã lưu khỏi máy Đổi tài khoản
Ví dụ

Use cases thực tế

Từ web app đến database — tunnel đúng loại cho đúng mục đích. 

# Web app dev — subdomain cố định (sau khi login)
htn-tunnel http 3000:myapp     # → https://myapp.33.id.vn

# API server
htn-tunnel http 8080:api       # → https://api.33.id.vn

# Database remote access (TCP)
htn-tunnel tcp 5432            # → tcp://33.id.vn:xxxxx

# SSH tunnel
htn-tunnel tcp 22              # → tcp://33.id.vn:yyyyy

# Quick share — không cần đăng ký (hết hạn 2h)
htn-tunnel http 3000           # → https://abc12xyz.33.id.vn
Self-Host

Chạy server của riêng bạn

Cần một VPS, một domain, và 3 bước. Wizard tự lo phần còn lại.

Bước 01

Tải binary server

Download htn-server cho hệ điều hành của VPS từ GitHub Releases.

# Linux x64
curl -Lo htn-server \
  https://github.com/nhh0718/htn-tunnel/releases/latest/download/htn-server-linux-amd64
chmod +x htn-server
sudo mv htn-server /usr/local/bin/
Bước 02

Chạy wizard setup

Interactive wizard cấu hình domain, SSL, nginx, tạo systemd service — tự động hoàn toàn.

sudo htn-server init
# Nhập domain, Cloudflare API key
# Wizard tự cài nginx + SSL + service
Bước 03

Chia sẻ với team

Server đã chạy. Đồng đội chỉ cần setup địa chỉ server của bạn là dùng được ngay.

# Trên máy đồng đội
htn-tunnel setup
# Nhập: your-domain.com:4443

Server CLI Reference

Lệnh Mô tả
htn-server init Interactive wizard: cấu hình domain, SSL, nginx, systemd
htn-server start Khởi động tunnel server
htn-server status Xem trạng thái server và các tunnel đang hoạt động
htn-server upgrade Tự động update server lên phiên bản mới nhất
htn-server backup Tạo backup toàn bộ dữ liệu (users, keys, config)
htn-server restore Restore từ backup — dùng khi migrate server
